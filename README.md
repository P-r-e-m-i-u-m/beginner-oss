## 🤖 How it works

Every repo in this directory passed an automated quality check:

| Check | What it means for beginners |
|-------|----------------------------|
| ✅ README | You can understand what the project does |
| ✅ CONTRIBUTING.md | You know exactly how to contribute |
| ✅ OSS License | You can legally contribute |
| ✅ 10+ contributors | Active community, not a ghost repo |
| ✅ Active in 90 days | Maintainer will actually review your PR |
| ✅ Open good first issues | Real tasks waiting for you right now |

**Weekly re-check:** Every Monday, the bot re-validates every listed repo. Dead repos get auto-removed.

## ➕ Suggest a repo

See a repo that's genuinely beginner-friendly but isn't listed here?

1. Fork this repo
2. Create `suggestions/owner-reponame.md` with the repo URL and why it's beginner-friendly
3. Open a PR

The bot validates it automatically. Pass = instant merge. Fail = PR closed with exact reasons.
